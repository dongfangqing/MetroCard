2、等待wifi生成热点 ，用户名是:XZSXT  密码是: 123456789

3、手机连上热点 ， 然后打开APP进行连接STM32 进行通信 

4、打开串口助手，输入 : #钱  ----> 确定发送   -----> 然后刷卡，在手机上也可以看到额度

5、输入 : $钱  ----> 确定发送   -----> 然后刷卡，在手机上也可以看到额度
